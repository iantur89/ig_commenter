## üöÄ Quick Start Guide

### Step 1: Download the Extension Files

1. **Download the extension** by clicking the green "Code" button at the top of this page
2. Click **"Download ZIP"** 
3. **Extract the ZIP file** to a folder on your computer (like Desktop or Documents)
4. **Remember where you saved it** - you'll need this location in Step 3

### Step 2: Get Your OpenAI API Key

1. Go to [OpenAI's website](https://platform.openai.com/api-keys)
2. **Sign up or log in** to your OpenAI account
3. Click **"Create new secret key"**
4. **Copy the key** (it starts with "sk-") and save it somewhere safe
5. **You'll need this key** to use the extension

### Step 3: Install the Extension in Chrome

1. **Open Google Chrome**
2. Type `chrome://extensions/` in the address bar and press Enter
3. **Turn on "Developer mode"** (toggle switch in the top-right corner)
4. Click **"Load unpacked"** button
5. **Navigate to the folder** where you extracted the extension files
6. **Select the folder** and click "Select Folder"
7. The extension should now appear in your Chrome extensions list

### Step 4: Pin the Extension (Optional but Recommended)

1. Click the **puzzle piece icon** in Chrome's toolbar
2. Find **"IG Comment Generator"** and click the **pin icon** üìå
3. The extension icon will now appear in your toolbar for easy access

## üì± How to Use

### First Time Setup

1. **Click the extension icon** in your Chrome toolbar
2. **Enter your OpenAI API key** in the "API Key" field
3. Click **"Save API Key"** - you should see a green success message
4. **Expand "System Instructions"** if you want to customize the AI's commenting style

### Generating Comments

1. **Go to any Instagram post** (not just the main feed - click on a specific post)

3. **Click "Generate AI Comment"**
4. **Wait for processing** (takes 10-30 seconds)
5. **Review the generated comment** in the Instagram comment field
6. **Edit if needed** and click "Post"

- **Comment quality**: Substantial, educational comments (2+ sentences) with coaching insights
- **Comment injection**: Automatically fills the comment field with a green border indicator
- **Success notification**: Green notification appears when comment is generated

## üîß Troubleshooting

### Extension Won't Load
- **Make sure Developer mode is ON** in `chrome://extensions/`
- **Check that you selected the correct folder** (should contain manifest.json)
- **Try refreshing the page** and reloading the extension

### "Could not establish connection" Error
- **Refresh the Instagram page** and try again
- **Make sure you're on a specific post** (URL should contain `/p/` or `/reel/`)
- **Check that the extension is enabled** in `chrome://extensions/`

### API Key Issues
- **Verify your API key** starts with "sk-"
- **Check your OpenAI account** has sufficient credits
- **Make sure the key is saved** by clicking "Save API Key"

### Comments Too Short
- **Use "Force Reload from File"** button in System Instructions
- **Check that instructions were saved** properly
- **Try generating a few comments** to see the AI adapt

### Comments Not Educational Enough
- **Edit the System Instructions** to emphasize educational value
- **Add specific coaching expertise** you want to highlight
- **Use the examples** in the instructions as templates
